sa-tests-db

Репозиторий содержит наборы тестов для статических анализаторов кода.

Тесты подобраны для реализации этапа "Домашнее задание" испытаний открытых и коммерческих статических анализаторов исходных кодов компилируемых и динамических языков программирования под патронажем ФСТЭК России [1].

Комплект тестов этапа «Домашнее задание» доступен всем участникам испытаний, а оценка результатов будет носить декларативный характер [2]. Каждая организация-участник, получив комплект тестов и, проведя испытания самостоятельно, также самостоятельно заполнит типовую анкету зафиксировав там результаты, полученные в ходе анализа комплекта тестов. Данные результаты будут переданы жюри и организаторам испытаний без какой-либо дополнительной верификации — цель данного этапа это не только верхнеуровневая оценка возможностей инструментов, но и выравнивание позиций и восприятия коллективами участников и жюри.

В критерии оценки этапа «Домашнее задание» войдут количественные показатели обнаружения заложенных в атомарные тесты дефектов кода, а также некоторые качественные характеристики инструментов, такие как маппинг детекторов на «пункты» ГОСТ 71207-2024, наличие документации по всем типам ошибок и выдача результатов разметки в открытых форматах.

[1] https://ib-bank.ru/bisjournal/news/21667 - инициализирующий пресс-релиз со списком участников испытаний и членов жюри

[2] https://ib-bank.ru/rbpos - пресс-релиз №1 с общим описанием целей, характеристик и плана испытаний


Структура директорий репозитория содержит 2 комплекта тестов, т.н. "Основной комплект" (basic_set) и "Дополнительный комплект" (additional_set).

```
sa-tests-db
    ├── additional_set
    ├── basic_set
    └── documentation
```
